    A S4L (Skate for Life) é uma plataforma online dedicada aos entusiastas do skate e a novas pessoas que ainda nem sabem que são entusiastas.
    Ofereceremos uma calculadora de peso personalizada para ajudar os skatistas a escolherem a configuração ideal de skate com base em seu peso corporal
    além disso irá calcular qual seria seu peso ideal e auxiliar na perda de peso dele utilizando o skate para obter uma vida mais saudável.

    O projeto utiliza HTML, CSS e JavaScript.
    HTML define a estrutura e o conteúdo,
    CSS controla o estilo e o design,
    e JavaScript adiciona interatividade e funcionalidade dinâmica.
    Juntos, essas três tecnologias
    formam a base para a criação de páginas web
    modernas e eficazes.
    E integra as APIs externas Web-data-viz
    para cadastro de informações
    e a Data-acqua-ino
    para fornecer dados em forma de gráficos.
    Todas as API's foram feitas por meio de Node.Js.
